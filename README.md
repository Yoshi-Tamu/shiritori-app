# しりとり

## 概要

ブラウザ上で動作するシンプルなしりとりのゲームです。\
主にサーバー側でルールを管理していて、ユーザーはフロントエンドから操作します。

---

## デプロイURL

https://URL追加予定

---

## 実装機能/デザイン

### 基本的な機能

- 直前の単語の表示
- 単語の入力機能
- しりとりルール判定（末尾と先頭の一致）
- 「ん」で終わる単語の入力禁止（ゲーム終了）
- 過去に使用した単語の入力禁止（ゲーム終了）
- ゲーム終了状態の管理
- リセット機能

---

### 追加した機能

- 過去履歴の表示
- 入力した回数の可視化
- ひらがな以外の入力を禁止

---

### デザイン

- すべての機能を1画面でわかりやすく配置
- ゲーム終了時はわかりやすいように入力フォームと送信ボタンのデザインを変更

---

## 工夫した点

- サーバー側でゲームの状態（gameOver）を管理
- ゲーム終了時にリセット以外の操作を受け付けないようにする
- 回数と履歴の表示でここまでのゲームの流れを振り返れるように
